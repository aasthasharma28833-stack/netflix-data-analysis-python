# netflix-data-analysis-python
This project analyzes the Netflix dataset using Python to discover
content trends, genre popularity, and country distribution.
## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
## Dataset
Netflix Movies and TV Shows dataset from Kaggle.
## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Data Visualization
4. Insights generation
## Key Insights
- Most Netflix content is released after 2015
- United States has the highest content
- Drama and Comedy are most popular genres
charts.png

1. Download dataset

3. Run all cells
